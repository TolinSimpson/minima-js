Why MinimaJS?

Modern web development shouldn't require massive frameworks and complex build processes. MinimaJS proves that powerful, reactive web applications can be built with minimal overhead and maximum developer productivity.

🎯 Minimalist Philosophy

Every feature is carefully considered. No bloat, no unnecessary abstractions - just the essentials for building modern web apps.

⚡ Instant Development

No build step, no compilation, no configuration. Just include the script and start building. Hot reload keeps you in the flow.

🌍 Universal Compatibility

Works in every modern browser without polyfills or transpilation. ES6+ baseline ensures broad compatibility.

Getting Started

Get MinimaJS up and running in less than 2 minutes:

Option 1: CDN (Fastest)

<!-- Include MinimaJS from CDN --> <script src="https://cdn.jsdelivr.net/npm/minimajs/src/minima.js"></script> <!-- For development (includes dev tools) --> <script src="https://cdn.jsdelivr.net/npm/minimajs/src/minima.dev.js"></script>

Option 2: Local Development

# Clone the repository git clone https://github.com/minimajs/minimajs.git cd minimajs # Start development server npm run dev # Open http://localhost:8080

Your First Component

const HelloWorld = MinimaJS.defineComponent(function(props) { // Initialize reactive state this.state = MinimaJS.createState({ count: 0 }); // Event handler const increment = () => { this.state.count++; }; // Return template return ` <div> <h1>Hello, ${props.name || 'World'}!</h1> <p>Count: ${this.state.count}</p> <button onclick="increment()">Click me</button> </div> `; }); // Render to DOM MinimaJS.render(HelloWorld, document.getElementById('app'));

Core Features

🧱 Component-Based Architecture

Build UIs with reusable, encapsulated components that manage their own state and lifecycle.

MinimaJS.defineComponent(componentFunction)

Creates a new component with automatic state management and lifecycle hooks.

const MyComponent = MinimaJS.defineComponent(function(props) { // Component state - automatically reactive this.state = MinimaJS.createState({ isVisible: true, items: [] }); // Lifecycle hooks this.onInit = function() { console.log('Component initialized'); }; this.onUpdate = function() { console.log('Component updated'); }; // Event handlers const toggleVisibility = () => { this.state.isVisible = !this.state.isVisible; }; return ` <div style="display: ${this.state.isVisible ? 'block' : 'none'}"> <h2>${props.title}</h2> <button onclick="toggleVisibility()">Toggle</button> </div> `; });

🔄 Reactive State Management

State changes automatically trigger UI updates. No manual DOM manipulation required.

MinimaJS.createState(initialState)

Creates a reactive state object using JavaScript Proxy.

// Local component state const state = MinimaJS.createState({ count: 0, name: 'John', todos: [] }); // State changes trigger re-renders automatically state.count++; // UI updates immediately state.todos.push({ id: 1, text: 'New todo' });

MinimaJS.store(name, initialState)

Creates or retrieves a global state store for sharing data across components.

// Create global store const userStore = MinimaJS.store('user', { name: 'Anonymous', preferences: { theme: 'light' } }); // Access from any component userStore.name = 'John Doe'; userStore.preferences.theme = 'dark';

🗺️ SPA Routing

Built-in router for single-page applications with dynamic parameters and History API integration.

MinimaJS.router.init(routes, container)

Initialize the router with route configuration.

const routes = [ { path: '/', component: HomePage }, { path: '/about', component: AboutPage }, { path: '/users/:id', component: UserPage }, { path: '/posts/:category/:slug', component: PostPage } ]; MinimaJS.router.init(routes, document.getElementById('app'));

MinimaJS.router.navigate(path)

Programmatically navigate to a new route.

// Navigate programmatically MinimaJS.router.navigate('/users/123'); // Components receive route parameters as props const UserPage = MinimaJS.defineComponent(function(props) { const userId = props.id; // From /users/:id route return `<h1>User ${userId}</h1>`; });

🛠️ Development Tools

Powerful development experience with hot reload, type checking, and debugging tools.

Runtime Type Checking (Dev Mode)

Optional type validation for props and state during development.

const TypedComponent = MinimaJS.defineComponent({ props: { title: String, count: Number, items: Array, isActive: Boolean, user: { id: Number, name: String, email: String } } }, function(props) { // Props are validated automatically in dev mode return `<div>${props.title}: ${props.count}</div>`; });

// Available in dev mode MinimaJS.dev.listComponents() // List all registered components MinimaJS.dev.getPerformanceStats() // View render performance MinimaJS.dev.inspectStore('user') // Inspect global stores MinimaJS.dev.getInternalState() // Framework internals

Complete API Reference

Core Methods

MinimaJS.render(component, container)

Renders a component into a DOM container.

  • component - Component function or instance
  • container - DOM element to render into

MinimaJS.createElement(tag, props, children)

Creates a virtual DOM node (VNode).

  • tag - HTML tag name
  • props - Element attributes and event handlers
  • children - Array of child VNodes or text

MinimaJS.onGlobalStateChange(storeName, key, callback)

Subscribe to global state changes.

  • storeName - Name of the store to watch
  • key - Specific property to watch
  • callback - Function called when value changes
  • Returns: Unsubscribe function

Router API

MinimaJS.router.getCurrentRoute()

Returns the current route information including parameters.

Route Configuration Format

{ path: '/users/:id', // Route pattern with parameters component: UserComponent // Component to render }

Development API (minima.dev.js only)

MinimaJS.dev.validateData(data, schema, path)

Manually validate data against a type schema.

Type Schema Format

{ name: String, age: Number, isActive: Boolean, items: Array, meta: Object, callback: Function, nested: { id: Number, title: String } }

Counter Example

const Counter = MinimaJS.defineComponent(function(props) { this.state = MinimaJS.createState({ count: props.initialCount || 0 }); const increment = () => this.state.count++; const decrement = () => this.state.count--; const reset = () => this.state.count = 0; return ` <div class="counter"> <h2>Count: ${this.state.count}</h2> <button onclick="increment()">+</button> <button onclick="decrement()">-</button> <button onclick="reset()">Reset</button> </div> `; }); MinimaJS.render(Counter, document.getElementById('app'));

Form with Validation

const ContactForm = MinimaJS.defineComponent(function(props) { this.state = MinimaJS.createState({ name: '', email: '', message: '', errors: {} }); const validateForm = () => { const errors = {}; if (!this.state.name.trim()) errors.name = 'Name is required'; if (!this.state.email.includes('@')) errors.email = 'Valid email required'; if (this.state.message.length < 10) errors.message = 'Message too short'; this.state.errors = errors; return Object.keys(errors).length === 0; }; const handleSubmit = () => { if (validateForm()) { alert('Form submitted successfully!'); } }; return ` <form onsubmit="event.preventDefault(); handleSubmit();"> <input type="text" placeholder="Name" value="${this.state.name}" oninput="this.state.name = this.value" /> ${this.state.errors.name ? `<div class="error">${this.state.errors.name}</div>` : ''} <input type="email" placeholder="Email" value="${this.state.email}" oninput="this.state.email = this.value" /> ${this.state.errors.email ? `<div class="error">${this.state.errors.email}</div>` : ''} <textarea placeholder="Message" oninput="this.state.message = this.value" >${this.state.message}</textarea> ${this.state.errors.message ? `<div class="error">${this.state.errors.message}</div>` : ''} <button type="submit">Send Message</button> </form> `; });
